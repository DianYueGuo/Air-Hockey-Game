# Decisions

- Goals are left/right; Player 1 defends left, Player 2 defends right.
- Mallets are constrained to their respective left/right halves of the table.
- Default dev window size is 960x540.
- Settings and calibration apply when entering Play; settings also refresh when resuming from Pause.
- License: MIT.
- Separate scoreboard window uses `pygame._sdl2` when available; otherwise fallback to HUD.
