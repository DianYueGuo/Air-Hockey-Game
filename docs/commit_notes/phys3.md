# Phys3

What changed
- Increased mallet restitution and density, and reduced mallet friction for stronger puck impacts.
- Increased default mallet speed limit.

Manual test steps
- Run `python3 -m air_hockey.main`.
- In Play, swing a mallet into the puck and confirm it travels farther.
- Adjust Mallet Speed in Physics Tuning if needed.

Known issues
- Existing settings may override defaults; update Physics Tuning to apply.
