# E1

What changed
- Added calibration data structures for per-player bounds.
- Implemented a guided calibration screen that captures left/right/top/bottom extremes per player.
- Live camera preview with detection markers is shown during calibration.

Manual test steps
- Run `python -m air_hockey.main`.
- Click Calibration and follow the prompts, pressing Capture for each extreme.
- Verify the status text updates as each step is recorded.

Known issues
- Calibration is not persisted yet; values are stored in memory only.
