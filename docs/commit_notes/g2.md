# G2

What changed
- Added settings persistence to the user data directory.
- Implemented basic Settings UI toggles for webcam view and scoreboard mode.
- Play screen now loads persisted settings (theme, webcam view, scoreboard mode, mallet speed).

Manual test steps
- Run `python -m air_hockey.main`.
- Open Settings and toggle webcam view and scoreboard mode, then exit.
- Restart the app and verify the choices persist in Play.

Known issues
- Only a subset of settings are exposed in the UI so far.
